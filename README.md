# ONEMANSHYO artist site app

**Live:** [onemanshyo.com](https://onemanshyo.com)

Single-file HTML portfolio for electronic music DJ/producer Wes Smith. WebGPU shaders, hash-based SPA routing, zero dependencies, mobile responsive. Built entirely with [Claude AI](https://claude.ai) as a development partner.

## Stack

- Single HTML file (~2,800 lines)
- WebGPU compute + render shaders (desktop)
- SPA with hash routing
- Mobile responsive (≤768px)
- No frameworks, no build process, no dependencies
